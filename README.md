# Next.js Template

Template write with the following dependencies:

- React
- Next.js
- TypeScript
- Eslint
- Prettier
- i18Next
- Styled Components
- Zustand
- PM2

## development

```bash
pnpm run dev
```

## format

```bash
pnpm run format
```

## build

```bash
pnpm run build
```

## production preview

```bash
pnpm run preview
```

## production

```bash
pnpm run start
```

## deploy

```bash
pnpm run deploy
```
